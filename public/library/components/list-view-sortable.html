List View — Sortable

Click any column header to sort ascending; click again to flip to descending. JS hook: wrap in [data-sortable-list], give the body [data-list-body], and put a .list-sort button in each header cell with data-key and data-type="string|number". Cells reference their key via data-cell="key"; numeric cells should also include data-value for the raw number.
