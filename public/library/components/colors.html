Foundation

Color Tokens

All colors are stored as raw HSL channels in CSS custom properties on :root, so they compose with opacity (e.g. hsl(var(--accent-warm) / 0.5)). Reference them via the utility classes (.bg-accent-warm, .text-accent) or directly in custom rules.

Files: library.css :rootFormat: H S% L%
