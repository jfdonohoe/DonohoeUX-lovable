Accordion

Click any row to expand/collapse. Multiple can be open at once.

JS hooks: data-accordion · .accordion-item · .acc-trigger · .acc-panel

Designing with empathy, building with purpose. Warmth over cleverness; clarity over density.

Reference any color with hsl(var(--token)), or use the utility classes like .bg-fill-blue-light and .text-accent.

Each component lives at /library/components/[name].html with a live demo and snippet you can copy.
